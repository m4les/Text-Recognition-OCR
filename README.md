# Text-Recognition-OCR
Recognizes text, it may also correct grammar.

This project uses PaddleOCR to accurately detect text from images with high accuracy. It's designed to be simple, portable, and fast. 
It first extracts the text with PaddleOCR, seperates the words, and then corrects grammar for higher accuracy. It works with screenshots, pngs, etc.
It may take 3 - 5 minutes to load upon startup.
